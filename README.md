# AGLC Citation Generator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A lightweight, client-side web app that builds Australian Guide to Legal Citation (AGLC4) references in seconds.
No sign-ups, no tracking — just clean citations.

---

## Features

- **20 source types** across five categories:
  - **Cases** — Domestic (Australian), International
  - **Legislation & Government Materials** — Legislation, Parliamentary Debates (Hansard), Written Submissions
  - **Secondary Sources** — Books, Journal Articles, Encyclopedia/Dictionary Entries, Theses/Dissertations, Reports
  - **Other Sources** — Newspaper Articles, Websites, Conference Papers, Speeches, Press/Media Releases, Interviews, Film/Television/Media, Social Media Posts, Written Correspondence
  - **International Materials** — Treaties
- **Pinpoint references** (page, paragraph, section) with toggle for applicable source types
- **Live preview** with smooth animations
- **Save / delete / export** citations to plain-text `.txt` file
- **LocalStorage only** — no server, no data leaves the browser
- **Flatpickr** date picker for AGLC-formatted dates
- **Particles.js** animated background
- Fully responsive, dark-mode friendly design
- First-load disclaimer with legal terms
- Self-contained privacy policy and equity statement

## Quick Start

```bash
git clone https://github.com/kaiwatanabe/AGLC4-GEN.git
cd AGLC4-GEN
# open index.html in your browser
```

No build tools or dependencies to install — it runs entirely in the browser.
